# Philip Tran - Personal Website

A clean, modern personal portfolio website showcasing my experience, education, skills, and projects.

## About

This is my personal website built with HTML, CSS, and JavaScript. It features a minimalist black and white design inspired by Palantir and Nightwing, emphasizing clean typography and thoughtful spacing.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Clean Aesthetic**: Black and white color scheme with smooth transitions and minimal design
- **Interactive Navigation**: Sticky navbar with smooth scrolling to sections
- **Timeline Visualization**: Professional experience displayed in a clean timeline format
- **Mobile-Friendly**: Hamburger menu for mobile navigation

## Sections

- **About**: Introduction and background
- **Education**: Academic credentials and coursework
- **Experience**: Professional work history with detailed descriptions
- **Skills**: Technical skills organized by category (Languages, Frameworks & Tools, Concepts)
- **Contact**: Ways to get in touch

## Technologies Used

- HTML5
- CSS3 (with CSS custom properties)
- Vanilla JavaScript
- GitHub Pages for hosting

## Local Development

To run this website locally:

1. Clone the repository:
```bash
git clone https://github.com/kyprua/Website.git
```

2. Navigate to the project directory:
```bash
cd Website
```

3. Open `index.html` in your browser:
```bash
open index.html
```

Or use a local server:
```bash
python -m http.server 8000
# Then visit http://localhost:8000
```
